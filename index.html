WHO ARE WE?

AWS User Group India is a volunteer based group that is united by community-led learnings and diversified by the wide variety of tools and technologies closely knit with Amazon Web Services, cloud computing technology, AWS design implementation and servicing, high scalability / performance computing, production use cases of AWS.

AWS Community Days are community-organized cloud education events, featuring technical discussions and demos led by expert AWS users and industry leaders from around the world.

Community-based learning is the best way to enhance the concepts being taught by connecting with enthusiasts in person, thereby sharing first-hand experiences and familiar, accessible examples.

All skill levels are welcome to help accomplish our motto of learning, sharing and networking. Let's join hands to empower this tech community to make it bigger and better!

Serverless Bot for Web Chat, SMS and Phone Calls

NOTE: WORKSHOP WILL BE CONDUCTED ONLINE
Those who attend the entire workshop, will get amazon vouchers equivalent to the workshop ticket amount.
Date Duration
17th Oct, 2:00PM 4 Hours

Description

Speakers :

  • Prashanth HN
  • Dhaval Nagar

    • Objective

    Get hands-on experience in building a Serverless application using AWS Services. In this session, we will create a Centralized Bot using AWS Lex that can be initiated via SMS (Amazon Pinpoint), Web (AWS S3), or Phone Calls (Amazon Connect).

    Note: We will be using the AWS US region for the session.

    AWS Serverless Services:

    • Lex
    • Pinpoint
    • SNS
    • S3
    • DynamoDB
    • Amazon Connect
    • Serverless Framework

    Workshop Content:

    • Setup Lex Bot, Build, and Test
    • Integrate Bot with Website hosted on S3 and test from Browser
    • Integrate Bot with Pinpoint SMS and test from User Phone
    • Integrate Bot with Amazon Connect and test using a Phone Call

    Pre-requisites:

    • Active Internet Connection
    • Active AWS Account
    • Basic understanding of AWS Services like IAM, S3, Lambda, and DynamoDB
    • There will be a minimum charge as we will need to acquire SMS Long-code.

    Pricing:

    Service Billing Usage Cost
    Lex No Cost No Cost
    S3 No Cost No Cost
    Pinpoint for SMS Acquire Phone Number for Inbound SMS Cost of sending US SMS
    Amazon Connect for Call Center Direct Inward Number

    Note: Amazon Connect is not available for AISPL accounts, so we can not use it from the Indian AWS account, but we will show the full integration and demo during the workshop.
